# Kata 7

Séptima kata de la parte práctica de Ingeniería del Software II

Partiendo de la kata anterior, el objetivo de esta nueva kata es empaquetar y distribuir la aplicación como un producto instalable. Para ello, se deberán utilizar las capacidades de Maven para la compilación y empaquetado del proyecto, junto con jpackage, la herramienta introducida en Java 14, que permite crear instaladores nativos para diferentes sistemas operativos (Windows, macOS, Linux).
